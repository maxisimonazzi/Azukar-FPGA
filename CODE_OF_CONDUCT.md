# English

This project follows a respectful and inclusive collaboration policy.

---

## Our Standards

We encourage:

- Respectful communication
- Constructive feedback
- Collaboration and knowledge sharing
- Helping newcomers

Unacceptable behavior includes:

- Harassment or discrimination
- Offensive language
- Personal attacks
- Disruptive behavior

---

## Enforcement

Maintainers may remove content or restrict participation if rules are violated.

---

## Scope

This applies to all project spaces:

- GitHub issues
- Pull requests
- Discussions
- Documentation

---

## Reporting

If you experience problems, contact the project maintainers via GitHub issues or private message.

---

This project follows the spirit of the Contributor Covenant.

# Español

Este proyecto sigue una política de colaboración respetuosa e inclusiva.

---

## Nuestros estándares

Fomentamos:

- Comunicación respetuosa
- Feedback constructivo
- Colaborar y compartir de conocimiento
- Ayudar a quienes recién comienzan

El comportamiento inaceptable incluye:

- Acoso o discriminación
- Lenguaje ofensivo
- Ataques personales
- Comportamiento disruptivo

---

Los mantenedores pueden eliminar contenido o restringir la participación si se violan las reglas.

---

- Issues de GitHub

- Discussions
- Documentación
